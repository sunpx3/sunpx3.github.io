搭建hexo中遇到的一些问题

2016-11-10 
deploy: 
 type: git
 repository: https://github.com/sunpx3/sunpx3.github.io.git
 branch: master

之前照着别人的教程搭建hexo , 到hexo d 提交代码到github上时报错，说没设置用户名。
百度后在 stackoverflow 上看到改成


deploy:
 type: git
 repository: git@github.com:sunpx3/sunpx3.github.io.git
 branch: master

成功解决。

http://localhost:4000
使用hexo s启动本地服务的时候不能访问，查了下端口，发现是4000端口被占用，故修改端口后访
hexo s -p 5000 ，成功解决。

错误:ERROR Deployer not found: git
解决:npm install hexo-deployer-git –save
